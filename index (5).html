iProspect — Fan-Out Intelligence
Your content ranked.
Will AI cite it?
Traditional search ranks pages. AI search synthesises answers — pulling from multiple sources, decomposing queries into sub-questions, and citing only the pages that cover each one well. This tool simulates that process so you can see exactly where your content stands.
What is query fan-out?
When someone asks an AI search engine a question, it doesn't just look for pages that match the query. It breaks the question apart — into a set of smaller, more specific sub-queries — and retrieves the best answer for each one. The final response is assembled from multiple sources, each cited for a different part of the answer.
That means your page might rank well for the main keyword but still be invisible in AI-generated answers, because it only covers one dimension of what the AI is actually looking for.
What this tool does
Simulates fan-out
For any page and seed query, we generate the sub-queries an AI would fan out to — based on real intent decomposition patterns.
Scores your coverage
Each sub-query is scored against your actual page content from the Screaming Frog extract. You see exactly where the gaps are.
Uses real GSC data
Upload your Google Search Console queries export and the tool maps real search demand to each page — pre-filling seed queries automatically.
Works per project
Every client lives in its own workspace. Data stays local in your browser. Nothing is sent anywhere except your Gemini API calls.
How to get started
01
Set up a project
Click "New project" in the sidebar. Give it a name, the client domain, and a note on tone.
02
Upload your Screaming Frog exports
Add the Custom JavaScript extractor CSV and/or the Internal All tab CSV. These give us the page content fingerprint for each URL.
03
Upload your GSC queries export
From Google Search Console, export the Queries report as CSV. The tool maps real queries to your pages and pre-fills seed queries automatically.
04
Review and adjust seed queries
Each page gets a suggested seed query — either from GSC (highest impression query) or from the page title. Edit any that don't look right.
05
Run the fan-out
Hit "Run all" or analyse pages individually. You'll see a coverage score, a map of all sub-queries, the gaps, and recommendations.
Before you start
You'll need a Gemini API key from Google AI Studio — free to get started. Paste it into the field in the top-right corner. It's stored only in your browser session.
For the GSC export: go to Search Console → Performance → Queries, set your date range, then click Export → Download CSV.
Your data stays in your browser. No account needed.
